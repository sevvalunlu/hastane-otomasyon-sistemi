# 🏥 Hastane Otomasyon Sistemi

Bu proje, bir hastanenin günlük işleyişini dijital ortama taşıyan, C# ve Microsoft SQL Server kullanılarak geliştirilmiş kapsamlı bir masaüstü otomasyon uygulamasıdır.

## 🚀 Proje Amacı
Hastaların randevu alma süreçlerini kolaylaştırmak, doktorların hasta takibini düzenlemek ve sekreterlerin sistem yönetimini tek bir merkezden, hızlı ve hatasız bir şekilde yapabilmesini sağlamaktır.

## 🛠️ Kullanılan Teknolojiler
* **Programlama Dili:** C#
* **Veritabanı:** Microsoft SQL Server
* **Arayüz:** Windows Forms (WinForms)

## 📌 Temel Özellikler (Modüller)

### 🧑‍⚕️ Doktor Modülü
* Doktor girişi ve profil yönetimi
* Gelen randevuları görüntüleme ve detaylarına erişim
* Hastalara ait geçmiş sağlık kayıtlarını inceleme
* Şikayet ve teşhis girişleri

### 👨‍💼 Hasta Modülü
* Yeni hasta kaydı oluşturma ve giriş yapma
* Poliklinik ve doktor seçerek randevu alma
* Geçmiş randevuları görüntüleme

### 📝 Sekreter (Yönetici) Modülü
* Sekreter girişi
* Yeni doktor kayıtları oluşturma ve branş atamaları
* Poliklinik/Branş ekleme ve düzenleme
* Genel randevu takibi ve onayı
* Sistemdeki genel duyuruları yayınlama
